## v1.0.2 发布记录 (2026-06-21)

**主题**: P0 阻塞项清零 — 全量验收修复

### Phase 1: 安全闸口与核心链路
- **Config**: `loadYAML` 实现 `gopkg.in/yaml.v3` 解析；`main.go` 使用 `config.Load()` 统一配置
- **Policy Engine**: 添加 `Condition` 字段匹配（`action_types`/`target_patterns`/`risk_min`），`Priority` 排序
- **Capability Engine**: 实现 `required_capabilities ⊆ declared_capabilities` 静态授权检查
- **Risk Engine**: 修正 `browser.open`(L1)、`browser.click`(L2)、`fs.delete`(L3)、`database.delete`(L4)、`payment.initiate`(L5) 风险等级
- **Token**: `IssueToken` 接入 `publishApproved`；`TokenClaims` 添加 `scope`/`issuer` 字段
- **Audit Engine**: ring buffer (1000条) + 独立 goroutine 异步批量刷盘到 `audit.log`
- **审批超时**: `time.AfterFunc(300s, ...)` 强制执行 → `ActionRejected("approval_timeout")`
- **异步审批竞态**: Scheduler 处理 `GoalPauseRequested`/`GoalRollbackRequested` 取消 pending；Governance `handleUserApproved` 检查超时/取消；`handleActionCancelled` 发布 `ActionRejected("state_changed")`

### Phase 2: 内核行为完整性
- **Action 状态机**: 新增 `ActionStatus` 9 状态（Pending→Scheduled→Approved→Executing→Verifying→Completed/Failed/Recovering）；Scheduler 所有 handler 中追踪
- **验证循环**: Scheduler 订阅 `VerificationResult` → PASS/FAIL → 自修正最多 3 次 → `SelfCorrectionExhausted`
- **Goal 状态矩阵**: 新增 `VerificationFailed→Running`、`SelfCorrectionExhausted→Failed(terminal)` 转换
- **seccomp 违规处理**: `handleActionFailed` 检查 `error_type=="seccomp_violation"` → 直接 HumanIntervention
- **GoalAnchor**: `New` 接受 `*GoalAnchorTracker`，`handleGoalCreated` 调用 `Increment`，达 N=20 时 `goal_anchor_check: true`

### Phase 3: 功能完整性
- **Plugin 签名验证**: `discovery.go` 实现 SHA256 签名验证（安全旁路消除 ❌→✅）
- **Plugin 发现**: `pluginrunner.go` 使用真实 `PluginDiscovery` 替代 `findPluginBinary` stub
- **MissionGraph 校验**: 拓扑排序（Kahn's algorithm）循环检测 + 边引用校验 + 自循环检测
- **Persona**: 移除审批消息 `⚠` emoji（安全硬约束）
- **Daemon API**: `health` 添加 `uptime`，`system/status` 添加 `port`/`uptime`，`log`/`events` 从 StateStore 查询
- **CLI**: 新增 `list`/`status`/`pause`/`resume`/`stop` 命令
- **Logger**: 改用 JSON 格式；`~/.goalos/` 根目录 0700

### 安全旁路清零
| 标记 | 修复前 | 修复后 |

| `W1: default grant` (governance.go:133) | Capability 永远 GRANTED | 子集检查强制 enforce |
